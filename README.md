# wmk-image

## Image Utility component for Gatsby projects

### Installation

```sh
npm install --save wmk-image
```

### Usage

```jsx
<WmkImage
  image={Img} // an instance of the Img class (from within library)
  crop="crop-size" // add crop sizes
  style={styleObject}
  className="class-string"
/>
```
